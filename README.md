ASTRO Camp 的JavaScript課程，小專案練習。
透過實際的寫一小段js的code，加深HTML的運作流程及原理。
這個專案透過javascript所撰寫的code控制分別按下兩顆按鈕時，會有對的結果顯示在中間的框框，
一個表示「增加」、一個表示「減少」。
至於每次增加多少及減少多少也可以透過javascript來控制。
